- Asynchronously sends data to a specified URL using HTTP PUT requests.
- Handles potential errors and logs the response status.
- Returns the status and response of the HTTP request.

~~~~python
    """
    Asynchronously sends a PUT request to a specified URL with the provided data payload.

    Args:
        url (str): The URL to send the PUT request to.
        data_payload (dict): The JSON payload to send in the PUT request.
        session (ClientSession): An aiohttp ClientSession object for making HTTP requests.
        id (int): A unique identifier for tracking the request.

    Returns:
        dict: A dictionary containing the request status, response data, and the request ID.
    """
        ~~~~

**`cleanup_old_files(files_in_folder)` Function**:
- Checks files in the specified folder and removes files older than one day.
- Sends a notification to Microsoft Teams when files are deleted.

~~~~python
def cleanup_old_files(files_in_folder):
    '''
    Check the modified date of the file and if the file is older than 1 day then remove the file and send a message to Teams.
    '''
    for file in files_in_folder:
        # Check if you have access to the file; if not, just skip the file
        if not os.access(file, os.R_OK):
            logging.info("No access to file: {}".format(file))
            continue
        # Check if the file is older than 1 day
        try:
            file_modified_date = os.path.getmtime(file)
            if (time.time() - file_modified_date) > 86400:
                # Remove the file
                os.remove(file)
                logging.info("File removed because it is older than 1 day: {}".format(file))
                # Send a message to Teams (commented out for now)
                # myTeamsMessage.text("File removed because it is older than 1 day")
                # myTeamsMessage.send()
        except FileNotFoundError:
            logging.info("File not found when trying to access: {}".format(file))
        except PermissionError:
            logging.info("No access to remove file: {}".format(file))
            ~~~~

- Asynchronously sends data chunks to the API using the `put()` function.
- Handles unsuccessful API responses by dumping the data into a pickle file for later review.
- Logs the operation duration and waits for a specified interval before repeating the process.
